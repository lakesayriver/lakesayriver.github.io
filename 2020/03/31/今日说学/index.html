自学

github+hexo搭建自己的博客

1. 安装nodejs

nodejs下载地址

2. 安装git，在github上创建repository,命名为lakesayriver.github.io
3. 安装hexo

会出现卡住现象，可能和网络有关系

4. hexo初始化一个博客
5. 下载主题

进入创建的博客目录下，下载一个主题，这里的主题为常用的极简主题

6. 配置

博客的配置文件为_config.yml，注意键值对冒号后面有个空格！如下为主要配置：
title: lakesayriver
author: lakesayriver
language: zh-Hans
theme: next
deploy:
type: git
repository: https://github.com/lakesayriver/lakesayriver.github.io.git

7. 编写博客，测试

在lakesayriver.github.io/source/_posts/下新建博客内容，用markdown来写，也算是学习了markdown的语法吧
hexo s 启动服务，浏览器输入localhost:4000访问编写的博客

8. 发布

编写后发布到github托管,首先安装自动部署发布工具

然后执行

第一次需要输入github账号密码
