脚本同步系统时间，要求：1.每30s同步一次时间；2.同步失败发送邮件；3.同步成功100次发送邮件

运行脚本时ctrl+c中断，kill进程方法:

$ jobs

同步系统时间：

$ /etc/init.d/xinetd stop
$ /etc/init.d/xinetd start
$ etstat -nltup|grep 37
$ rdate xxx.xxx.xxx.xxx

或者

$ ntp xxx.xxx.xxx.xxx

代码如下：

#!/bin/env bash
ntp=10.1.1.2
vcn=0
while true
do
ntp ntp &>/dev/null
  if[ $? -ne 0 ];then
    echo "system ntp date failed"|mail -s "check system date" root@localhost
  esle
    let vnc++
    if[ vnc -eq 100 ];then
      echo "system ntp date succeed"|mail -s "check system date" root@localhost && vcn=0
    fi
  fi
sleep 30 
done

案例

批量创建10个用户，要求：1.stu1-stu5的用户UID分别为1001-1005，stu6-stu10用户的家目录分别在/rhome/stu6-/rhome/stu10

$ useradd -u username 指定UID 
$ useradd -d username 指定目录
$ passwd 123456 --stdin username  设置密码

代码如下：

#!/bin/bash
[ -d /rhome ]|| mkdir /rhome
i=1
until [ i -gt 10 ]
do
  if [ i -le 5 ];then
    useradd -u $[1000+i] stu$i
    echo 123456|passwd --stdin stu$i
  else
    useradd -d /rhome/stu$i stu$i
  let i++
done
