shift

参数移位
语法：
shift num 左移num位
示例：

#!/bin/bash
sum=0
while [ $# -ne 0 ]
do 
let sum=$sum+$1
shift 
done
echo $sum
expect

扩展程序，tcl语言，实现自动应答
案例一：
远程登录到服务器上，不操作

#!/usr/bin/expect
set ip [ lindex $argv 0 ]
set pass [ lindex $argv 1 ]
set timeout 5
#开启一个程序
spawn ssh root@10.1.1.1
#捕获相关内容
expect{
    "yes/no?":{send "yes\r";exp_continue}
    "password":{send "123456\r"}
}
interact

案例二：
远程登录到服务器上，并进行一些操作

#!/usr/bin/expect
set ip [ lindex $argv 0 ]    #第一个位置参数
set pass [ lindex $argv 1 ]    #第二个位置参数
set timeout 5
#开启一个程序
spawn ssh root@10.1.1.1
#捕获相关内容
expect{
    "yes/no?":{send "yes\r";exp_continue}
    "password":{send "123456\r"}
}
expect "#"
send "rm -rf /tmp/*\r"
send "touch /tmp/test.log\r"
send "date +%F"
expect eof

案例三：
循环在指定主机上创建用户和文件(在shell脚本中使用expect)

> read ip pass < 1.txt        从1.txt中读取内容到变量中，一次只能读取一行
> while read ip pass;do echo $ip;echo $pass;done < 1.txt   读取1.txt的全部内容到变量中

代码：

#!/bin/bash
while read ip pass
do
    /usr/bin/expect <<-END &>/dev/null
    #开启一个程序
    spawn ssh root@$ip
    #捕获相关内容
    expect{
        "yes/no?":{send "yes\r";exp_continue}
        "password":{send "$pass\r"}
    }
    expect "#" { send "useradd hyj;rm -r /tmp/test.log\r" }
    expect eof
    END   #和前面的END呼应
    echo "$ip服务器上创建用户成功！"
done < 1.txt
数组

数组分类：
普通数组：以整数作为数组的索引下标
关联数组：可以以字符串作为数组的索引下标
数组的定义：
一次赋予一个值

数组名[索引下标]=值
array[0]=av1
array[1]=av2

一次赋予多个值

数组名=(var1 var2 var3)
array=(1 2 3 "hello" [8]=world)

数组读取

${数组名[索引下标]}
echo ${!array[@]} 获取数组元素的索引下标
echo ${#array[*]} 获取所有元素的个数
echo ${array[@]:1,2} 获取数组的从索引1 开始的后两个元素
查看普通数组信息
