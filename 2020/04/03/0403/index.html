RANDOM

随机产生0-32767之前的数

案例1：

产生一个phonenum文件，随机产生以139开头的手机号1000个，每个一行

sort -u 去重
let i++ 变量可以不加$

#!/bin/bash
file=./phonenum.txt
i=0
while [ $i -lt 1000 ]
do
    j=0
    phone=139
    while[ $j -lt 8 ]
    do
        phone=$phone$[$RANDOM%10]
        let j++
    done
    echo $phone >> file
    let i++
done
案例2：

在上面生成的1000个手机号中随机抽取五个幸运观众,显示前三位和后四位

wc -l filename 打印文件行号和内容
a=12345678;${a:3:4} 获取变量a的第四位开始取4位数字
sed -i 直接在文件上编辑
sed -i ‘n,md’ filename 删除filename文件的第n到m行
sed -i “${var}d”
把执行命令赋值给变量 变量=cmd

脚本代码

#!/bin/bash 
file=./phonenum.txt
i=0
whie [ $i -le 4 ]
do
 line=`wc -l $file|cut -d' ' -f1`
 luck_line=$[$RANDOM%$line+1]
 luck_one=`head -$luck_line $file|tail -1`
 #显示到屏幕
 echo "139****${luck_one:7:4}"
 #删除抽中的号码
 sed -i "/$luck_line/d" file
 let i++
done
