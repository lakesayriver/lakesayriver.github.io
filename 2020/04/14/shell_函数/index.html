函数

发表于 2020-04-14 | 分类于 shell

1.函数定义

方法一：

函数名()
{
    函数体
}

方法二：

function 函数名()
{
    函数体
}

return 结束函数，指定返回值

2.函数调用

临时调用(只在当前终端能够调用)

  • source 函数.sh

    定义到用户环境变量中（环境变量可以在/etc/profile或~/.bashrc中定义）

    # vim ~/.bashrc
menu()
{
  cat <<-END
  1   tea
  2   coffee
  3   lemon
  END
}

    定义到全局变量

  • vim /etc/bashrc

    脚本中调用函数

  • 脚本中定义一个函数，直接调用

    函数名 参数

  • 脚本中加载一个函数

    source function.sh

3.案例

需求

  • 编写一个脚本收集用户的信息（姓名，年龄，性别）,如不输入一直提示用户输入；
  • 根据收集的用户信息输出对应内容

代码


#!/bin/bash
declare -A info

read -p "请输入您的姓名：" name
while [ -z $name ]
do
read -p "请输入您的姓名：" name
done
info["姓名"]=$name

read -p "请输入您的性别：" gender
while [ -z $gender ]
do
read -p "请输入您的性别：" gender
done
info["性别"]=$gender
read -p "请输入您的年龄：" age
while [ -z $age ]
do
read -p "请输入您的年龄：" age
done
info["年龄"]=$age

#打印信息
for i in ${!info[@]}
do
    echo 您的$i是：${info[$i]}
done
  • 优化版
    #!/bin/bash
function input()
{
      read -p "$1" input
      if [ -z $input ];then
        input $1
      fi
      echo $input
}
declare -A info

info[“姓名”]=input 请输入您的姓名：
info[“性别”]=input 请输入您的性别：
info[“年龄”]=input 请输入您的年龄：
#打印信息
for i in ${!info[@]}
do
echo 您的$i是：${info[$i]}
done
```
