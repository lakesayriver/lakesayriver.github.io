案例三：
循环在指定主机上创建用户和文件(在shell脚本中使用expect)

> read ip pass < 1.txt        从1.txt中读取内容到变量中，一次只能读取一行

#!/bin/bash
while read ip pass
do
    /usr/bin/expect <<-END &>/dev/null
    #开启一个程序
    spawn ssh root@$ip
    #捕获相关内容
    expect{
        "yes/no?":{send "yes\r";exp_continue}
        "password":{send "$pass\r"}
    }
    expect "#" { send "useradd hyj;rm -r /tmp/test.log\r" }
    expect eof

    echo "$ip服务器上创建用户成功！"
done < 1.txt
数组

数组分类：
普通数组：以整数作为数组的索引下标
关联数组：可以以字符串作为数组的索引下标
数组的定义：
一次赋予一个值

数组名[索引下标]=值
array[0]=av1
array[1]=av2

一次赋予多个值

数组名=(var1 var2 var3)
array=(1 2 3 "hello" [8]=world)

数组读取

${数组名[索引下标]}
echo ${!array[@]} 获取数组元素的索引下标
echo ${#array[*]} 获取所有元素的个数
echo ${array[@]:1,2} 获取数组的从索引1 开始的后两个元素
查看普通数组信息
declare -a

RANDOM

随机产生0-32767之前的数

案例1：

产生一个phonenum文件，随机产生以139开头的手机号1000个，每个一行

sort -u 去重
let i++ 变量可以不加$

#!/bin/bash
file=./phonenum.txt
i=0
while [ $i -lt 1000 ]
do
    j=0
    phone=139
    while[ $j -lt 8 ]
    do
        phone=$phone$[$RANDOM%10]
        let j++
    done
    echo $phone >> file
    let i++
done
案例2：

在上面生成的1000个手机号中随机抽取五个幸运观众,显示前三位和后四位

wc -l filename 打印文件行号和内容
a=12345678;${a:3:4} 获取变量a的第四位开始取4位数字
sed -i 直接在文件上编辑
sed -i ‘n,md’ filename 删除filename文件的第n到m行
sed -i “${var}d”
把执行命令赋值给变量 变量=cmd

脚本代码

#!/bin/bash 
file=./phonenum.txt
i=0
whie [ $i -le 4 ]
do
 line=`wc -l $file|cut -d' ' -f1`
 luck_line=$[$RANDOM%$line+1]
 luck_one=`head -$luck_line $file|tail -1`
 #显示到屏幕
 echo "139****${luck_one:7:4}"
 #删除抽中的号码
 sed -i "/$luck_line/d" file
 let i++
done

或者

代码如下：

#!/bin/env bash
ntp=10.1.1.2
vcn=0
while true
do
ntp ntp &>/dev/null
  if[ $? -ne 0 ];then
    echo "system ntp date failed"|mail -s "check system date" root@localhost
  esle
    let vnc++
    if[ $vnc -eq 100 ];then
      echo "system ntp date succeed"|mail -s "check system date" root@localhost && vcn=0
    fi
  fi
sleep 30 
done

语法结构

do

案例

批量创建10个用户，要求：1.stu1-stu5的用户UID分别为1001-1005，stu6-stu10用户的家目录分别在/rhome/stu6-/rhome/stu10

$ useradd -u username 指定UID 
$ useradd -d username 指定目录
$ passwd 123456 --stdin username  设置密码

代码如下：

#!/bin/bash
[ -d /rhome ]|| mkdir /rhome
i=1
until [ i -gt 10 ]
do
  if [ i -le 5 ];then
    useradd -u $[1000+i] stu$i
    echo 123456|passwd --stdin stu$i
  else
    useradd -d /rhome/stu$i stu$i
  let i++
done
